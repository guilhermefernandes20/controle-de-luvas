# Projeto de Controle de Luvas

Sistema simples para controle da produção de luvas.

## Funcionalidades
- Salvar marcação diária
- Calcular quantidade e valor total
- Excluir registros
- Armazenar sempre os dados

## Como executar
1. Baixe o código.
2. Compile com:
   ```bash
   javac ControleDeLuvas.java
